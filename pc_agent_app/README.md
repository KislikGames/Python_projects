# PC Agent App (Windows 10)

Отдельное приложение (в отдельной папке), чтобы не трогать ваши существующие файлы.

## Что умеет MVP
- Принимает цель пользователя.
- Планирует шаги (базово).
- Выполняет «скиллы»:
  - `solve_wordle` — открывает Wordle в браузере и предлагает стратегию.
  - `earn_100_rub` — запускает безопасный workflow с обязательным подтверждением и реквизитами.
- Делает самооценку после задачи и сохраняет уроки в локальную SQLite-память.

## Важно
Это безопасный MVP: он **не делает денежные переводы автоматически** и не выполняет рискованные действия без подтверждения пользователя.

## Установка (Windows 10)
```bash
cd pc_agent_app
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
# если нужен авто-браузер Playwright:
python -m playwright install
```

## Запуск GUI
```bash
python main.py
```

## Тесты
```bash
python -m unittest discover -s tests -v
```
