# General Desktop Agent (Windows 10)

Обновлённая, более опрятная версия: универсальный агент, где задачи **не зашиты жёстко** в код (Wordle/100 руб были демо в прошлой версии).

## Что изменилось
- Вместо набора фиксированных скиллов теперь есть общий контур:
  1. Policy-проверка безопасности.
  2. LLM-планирование (через OpenAI API).
  3. Выполнение действий через инструменты (`create_text_file`, `open_url`).
  4. Self-review и сохранение в память (SQLite).
- Если API-ключ не задан, работает fallback-планировщик (ограниченный, но рабочий).

## Структура (опрятно и минимально)
- `app.py` — GUI приложение.
- `desktop_agent.py` — оркестратор агента.
- `llm.py` — работа с LLM (план + review).
- `tools.py` — инструменты ОС/браузера.
- `policy.py` — ограничения и подтверждения.
- `memory.py` — локальная память (SQLite).
- `config.py` — загрузка `.env`.

## Куда класть API-ключи
1. Скопируйте `.env.example` в `.env`:
   ```bash
   copy .env.example .env
   ```
2. Откройте `.env` и укажите ключ:
   ```env
   OPENAI_API_KEY=sk-...
   OPENAI_MODEL=gpt-4.1-mini
   AGENT_SANDBOX_DIR=
   ```

> `AGENT_SANDBOX_DIR` (опционально): если указать, файловые действия будут писаться в эту папку для безопасного теста.

## Установка
```bash
cd pc_agent_app
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## Запуск
```bash
python app.py
```

## Примеры задач
- "Создай текстовый файл на рабочем столе и запиши в него приветствие"
- "Открой Wordle в браузере"
- "Открой google.com"

## Ограничения текущей версии
- Инструменты пока базовые (`create_text_file`, `open_url`).
- Для сложных задач (игры, многошаговые UI-сценарии) нужно добавить новые инструменты (скриншоты, управление мышью/клавиатурой, vision-loop).
